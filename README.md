# Impulso Estratégico CX - Rediseño

Sitio estático para el rediseño de `impulsoestrategicocx.com`.

## Estructura

- `site/`: archivos listos para publicar.
- `site/index.html`: home principal.
- `site/styles.css`: estilos sin framework externo.
- `site/main.js`: menú mobile y envío del formulario.
- `site/assets/`: imágenes optimizadas.
- `brief.md`: objetivo, alcance y decisiones del proyecto.

## Staging recomendado

Publicar el contenido de `site/` en:

`https://impulsoestrategicocx.com/new/`

El formulario está preparado para llamar a `../enviar_formulario.php` cuando esté desplegado bajo `/new/`.
